# Cloudflare DNS Aggregator

A comprehensive hub for managing GitHub Pages deployments with custom domains via Cloudflare.

## Features

### 📍 Sites Tab
- Lists all deployed GitHub Pages sites
- Shows GitHub Pages URLs
- Displays custom subdomain mappings
- Quick links to visit sites and view repositories
- Add new site information easily

### 📚 Documentation Tab
- Complete beginner-friendly guide for Cloudflare DNS setup
- Step-by-step instructions with examples
- Troubleshooting section for common issues
- Quick reference table for DNS records
- Professional tips and best practices

## What's Included

- **Dashboard Interface** - Professional, easy-to-use tabs for managing sites
- **Detailed Documentation** - Complete guide for non-technical users
- **Site Registry** - Keep track of all your deployed sites
- **DNS Quick Reference** - Helpful table of DNS records

## How to Use

1. **Visit the Sites Tab** to see all deployed GitHub Pages sites and their custom domains
2. **Visit the Documentation Tab** for step-by-step instructions on how to:
   - Set up Cloudflare for DNS management
   - Create CNAME records for custom subdomains
   - Configure GitHub Pages with custom domains
   - Troubleshoot common issues

3. **Add New Sites** - Edit the HTML to add new sites to the registry
